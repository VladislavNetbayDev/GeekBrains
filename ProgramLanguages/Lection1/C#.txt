Console.WriteLine("Hello, World!")

Console - позволяет работать с окном терминала. Вводим распечатывать и вводить данные в программу через терминал

WriteLine - Выводить что то из терминала в текст

Команды:

dotnet new console - создать новый проект

dotnet run - запустить проект

write (); - вывод в одну строку

WriteLine(); - переход на новую строку

ReadLine(); - Считать строку из терминала

dotnet new gitignore - создание файла gitignore

int - целое

double - вещественное
